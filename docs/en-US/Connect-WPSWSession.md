# Connect-WPSWSession

## SYNOPSIS
Connects to wilma site and  selects primusquery credentials

## SYNTAX

```
Connect-WPSWSession [[-site] <String>] [<CommonParameters>]
```

## DESCRIPTION
Connect to wilma site, establish websession and selects primusquery credentials to use.
